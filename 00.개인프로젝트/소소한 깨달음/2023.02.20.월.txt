

display : flex 준 박스에 
height 를 주지 않으면 
이미지가 보이지 않는다. 


html 

    <div class="bg1 bx1">
          <a class="abc"> </a>
          <a class="yuja"> </a>
    </div>

main.css
.bx1 {
    display: flex;
    height: 100%;    !!! 여기를 뺴먹었는데, 화면에 백그라운드 이미지가 나타나지 않았음
}

.bx1 a {
    flex: 1;
}

.abc {
    background: url(../img/ABC쥬스.jpg) no-repeat top/50vw 100%;
}

.yuja {
    background: url(../img/yuja.jpg) no-repeat top/50vw 100%;
}
